# game_application
Java based game build on pure OOPs.

This is an street fighter game 
2 player 
where player 1 is cammy and player 2 is ken
moves for cammy:-
walk left : A
walk right: D
puch : F
jump: SPACE
kick: C


moves for Ken:-
walk left: left button 
walk right: right button
puch : P
jump: L
kick: K
